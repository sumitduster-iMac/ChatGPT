# UI Design Guide

## Color Palette

- **Primary Green**: `#10a37f` - ChatGPT brand color
- **Dark Background**: `#0d0d0d` - Main app background
- **Secondary Background**: `#1a1a1a` - Cards and panels
- **Text Primary**: `#ffffff` - Main text
- **Text Secondary**: `rgba(255, 255, 255, 0.6)` - Secondary text

## Typography

- **Font Family**: System fonts (-apple-system, SF Pro, Segoe UI)
- **Headings**: Semi-bold, 600 weight
- **Body**: Regular, 400 weight

## Components

### Title Bar
- Custom dark title bar with navigation controls
- Back/Forward/Reload buttons
- URL display with lock icon

### Loading Screen
- Centered logo with pulse animation
- Circular spinner
- Gradient background

## Responsive Design

The app adapts to window sizes with:
- Minimum width: 800px
- Minimum height: 600px
- Fluid layout for content area
