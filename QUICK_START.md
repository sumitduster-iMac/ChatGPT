# Quick Start Guide

## Electron App (Cross-Platform)

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Running the App
```bash
npm install
npm start
```

### Building for Distribution
```bash
# Build for your current platform
npm run build

# Build for specific platforms
npm run build:mac
npm run build:win
npm run build:linux
```
